Nordic CSS

A simple, useful, aesthetic, and deliberate CSS framework for building a cleaner web.

Installation

Install with npm install @kevin-wynn/nordiccss@0.1.0

Reference in your application's html like:

<link rel="stylesheet" href="nordic.css" />

The Grid

The grid is built upon a very simple basis of a 12 column grid. Using the style of grid-* you can replace each grid with 1-12 for your column size. Each grid has no padding and no margins but comes completely mobile friendly. We will discuss padding and margins in the next section.

Note: Grids must be placed within a grid-container and nested within a row div. For example: 

      <div class="grid-container">
    <div class="row">
        <div class="grid-6"></div>
    </div>
</div>

Margin and Padding

Margins and padding are set up to give a very white space oriented feel. Open and airy is ideal here in order to dray more attention to the actual content. There are 3 different sizes for each margin and padding; small, medium, and large.

Note: You can also define specific margin and padding directions. For example: margin-bottom-small or margin-right-medium, etc...

Margin

margin-small 12px

margin-medium 24px

margin-large 36px

Padding

padding-small 12px

padding-medium 24px

padding-large 36px

Spacers

Nordic CSS comes with a few basic spacers that can be used to space out content. Allowing for even more white space between sections. You can also double these spacers with background colors to add a nice color block.

Spacers come in a variety of sizes as well.

<div class="spacer-10"></div>
<div class="spacer-25"></div>
<div class="spacer-50"></div>
<div class="spacer-75"></div>
<div class="spacer-100"></div>

Forms and Buttons

Nordic CSS has a few stylings for forms and buttons, utilizing some of the colors available for button classes. The forms utlize a few classes to define full width inputs or inline inputs. There are a few pre-defined classes for buttons such as submit and cancel. However you can define any of the available colors with syntax like so... 

      <div class="button-group">
  <div class="button green2">
    Submit
  </div>
</div>

And an example of a form:

      <form class="form-container">
  <p class="text-grey3">Welcome to Nordic CSS. This is an example form.</p>
  <div class="input-group margin-bottom-small">
    <div class="input">
      <label for="firstName">First Name</label>
      <input name="firstName" type="text" />
    </div>
    <div class="input error">
      <label for="lastName">Last Name</label>
      <span class="text-small text-red2">Error Message</span>
      <input name="lastName" type="text" />
    </div>
  </div>
  <div class="input">
    <label for="email">Email</label>
    <input name="email" type="text" />
  </div>
  <div class="button-container">
    <div class="button-group">
      <button class="button submit">Submit</button>
      <button class="button cancel">Cancel</button>
    </div>
  </div>
</form>
